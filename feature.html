Objetivos

Objetivo General

• Ofrecer el mejor servicio a cada uno de los clientes para lograr satisfacer sus necesidades.

• Aumentar la venta y producción del negocio para sobresalir del mercado.

• Ser el negocio más confiable para los clientes.

• Mejorar la producción de los productos para tener una mejor venta.

• Tener un mayor aumento en clientes para aumentar utilidades.

Objetivos específicos

• Abrir más sucursales en la zona para una mejor distribución de nuestros productos.

• Contratar más personal para todas las sucursales

• Tener un mayor número de proveedores para una mejor calidad de los insumos y materiales requeridos.
